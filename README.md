# 파일 이름 일괄 변경기 (Batch File Renamer)

여러 파일의 이름을 한 번에 바꿔서 ZIP으로 다시 내려받을 수 있는 간단한 웹 애플리케이션입니다.
같은 사이트에 제작자의 자기소개(이력) 페이지도 포함되어 있습니다.

## 페이지 구성
- `index.html` — 파일 이름 일괄 변경 도구 (메인)
- `resume.html` — 자기소개 / 이력 페이지
- `style.css` — 디자인
- `script.js` — 도구 동작 로직

## 특징
- 접두사 / 접미사 붙이기, 찾기·바꾸기, 순번 매기기, 대소문자 변경
- 모든 처리는 브라우저 안에서만 이루어져 파일이 서버로 올라가지 않습니다.

## 로컬에서 보기
이 폴더에서 `index.html` 파일을 더블클릭해 브라우저로 열면 됩니다.
